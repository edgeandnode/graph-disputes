## Usage
1. Copy this repository.
```
git clone https://github.com/gnokoheat/nextjs-ant-design-typescript
cd nextjs-ant-design-typescript/
```
2. Install dependencies
```
yarn install
```

3. Run app
``` bash
# Dev
yarn dev

# Production
yarn build
yarn start
```
